# Modular Sandwich Maker

A Python command-line sandwich ordering program built with modular OOP design for ITSC 3155.

## Project Structure

- **`data.py`** — Stores recipes (small, medium, large) and available resources.
- **`sandwich_maker.py`** — `SandwichMaker` class that checks ingredient availability and makes sandwiches.
- **`cashier.py`** — `Cashier` class that handles coin input and payment validation.
- **`main.py`** — Entry point that ties all modules together in an interactive ordering loop.

## How to Run

```
python main.py or locally in IDE
```

Follow the prompts to select a sandwich size and insert payment.
